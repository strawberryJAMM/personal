The Strawberry WebRing
Home Page


Welcome to the Strawberry WebRing Home Page. This WebRing is dedicated to bringing together all web sites with a recognizable connection to this most wonderful berry. Sites which qualify for inclusion in this WebRing include those dedicated to the dissemination of strawberry recipes, strawberry gardening, strawberry entertainment, and strawberry trivia. Companies and individuals that use the strawberry plant as a symbol or name are also welcome.By using the ring, you can travel from one strawberry web site to another until you eventually end up back on the page you started from. You can also jump randomly around the WebRing, travel the WebRing in reverse or list the URLs of the next few pages in the ring. It's a great way to find out all you can about Strawberries on the Net!

How Do I Sign My Site Up?

If you maintain a site which is related to strawberries in some significant way -- recipes, gardening, festivals, etc., or even if you use the strawberry as a significant symbol or name -- and would like to join the Strawberry WebRing, all you need to do is fill in the form below and add a small block of HTML code to the top or bottom of the page whose URL you submit.

While you can change the content of the code in the fragment below to better suit the needs of your page, in order to be accepted into the ring your code fragment must contain at least the following:

  • A link to the Strawberry WebRing home page (this page)
  • A link to the next page in the WebRing
  • A link to the previous page in the WebRing

This code fragment is what makes the ring work. If you can not add the needed section of code to the selected page, please do not apply. Sorry if this seems harsh, but without this code the ring will not work.

If you meet the requirements above, please grab the code fragments and images you will need (shown below, and fill out the form (also shown below) to join the Strawberry WebRing.

What Code Do I Need to Add?

Two code fragments are displayed below so that you can cut & paste it right from this web page. There are two versions: with graphics and without graphics.

All you need to do is:

  1. Insert the code below so it displays either at the top or the bottom of the selected page in your site.
  2. Where it says id=youridnumber, you replace youridnumber with the actual unique ID number assigned to you.
  3. Where it says youremailaddress, replace that with your real E-mail address.
  4. And where it says yourname, replace that with your name or the name of your business.

If you have chosen the With-Graphics version, you MUST download the graphics locally to your server (it is a Bad Idea (TM) to use graphics from someone else's server or web site). If you can't load the graphics locally, just use the Without-Graphics version.

To download the three graphic files required for the With-Graphics version, point your cursor over the image on the page and right-click with your mouse (or hold your mouse button down if you are on a Mac). A browser menu will ask you if you want to save the file as something. Save the "WebRing" image as "swr.gif". Do the same for the "Previous" and "Next" graphics and save them as "swrprev.gif" and "swrnext.gif". You should put them in the same directory as your selected page. (You can put them in a different directory as long as you update the IMG tag SRC attributes to point to the files in the directory you selected).
